# Calculator Plus App

This repository contains the **Calculator Plus App** developed as part of the Git assignment for Hero Vired.  
It demonstrates branching strategies, feature development, bug fixing, pull requests, and release management using Git and GitHub.

---

## Project Overview

The Calculator Plus App is a simple Python-based calculator that supports:

- Addition  
- Subtraction  
- Multiplication  
- Division  
- Square Root (added in Version 2)  
- Error handling for divide-by-zero  

This project was developed through multiple Git branches, following a proper Git workflow:  
`main` → `dev` → `feature/sqrt`.
